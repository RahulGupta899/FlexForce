# FlexForce
Muscle Training Application (React)

## Features
1. Exercise Searching 
2. Exercise Tutorial from youtube
3. Exercise Grouping by body part

## Tech Used
1. Reactjs
2. Rapid API
3. Css
4. Google Fonts
